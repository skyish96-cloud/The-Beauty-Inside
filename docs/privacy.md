# Privacy - 개인정보 처리 방침

## 개요
Beauty Inside는 사용자 프라이버시를 최우선으로 합니다.

---

## 핵심 원칙

### 1. 데이터 최소화
- 서비스 제공에 필요한 최소한의 데이터만 처리
- 불필요한 개인정보 수집 없음

### 2. 즉시 삭제
- 분석 완료 후 사용자 이미지 즉시 삭제
- 서버에 사용자 얼굴 데이터 저장 안함

### 3. 투명성
- 데이터 처리 과정 명확히 공개
- 사용자에게 충분한 정보 제공

---

## 데이터 처리 상세

### 수집하는 데이터
| 데이터 | 수집 여부 | 저장 여부 | 용도 |
|--------|----------|----------|------|
| 사용자 얼굴 이미지 | O (일시적) | ❌ | 분석용 (즉시 삭제) |
| 얼굴 임베딩 벡터 | O (일시적) | ❌ | 유사도 계산 (즉시 삭제) |
| 분석 결과 점수 | O | O (익명) | 서비스 개선 |
| 세션 ID | O | O (익명) | 서비스 운영 |
| IP 주소 | X | ❌ | - |
| 쿠키 | 최소 | 세션용 | 서비스 기능 |

### 수집하지 않는 데이터
- ❌ 사용자 신원 정보
- ❌ 위치 정보
- ❌ 기기 고유 식별자
- ❌ 연락처 정보
- ❌ 소셜 미디어 정보

---

## 데이터 생명주기

```
사용자 촬영
     │
     ▼
┌─────────────────┐
│  이미지 전송    │  ← 암호화 전송 (HTTPS/WSS)
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  서버 수신      │  ← 메모리에서만 처리
│  (RAM 임시저장) │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  분석 처리      │  ← 얼굴 감지 → 임베딩 추출 → 유사도 계산
│  (1-2초)       │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  결과 전송      │  ← 닮은 연예인 정보만 전송
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  즉시 삭제      │  ← 이미지, 임베딩 메모리에서 삭제
│  (자동)        │
└─────────────────┘
```

---

## 보안 조치

### 전송 보안
- HTTPS/WSS 암호화 통신
- TLS 1.3 지원

### 서버 보안
- 이미지 파일 저장 없음
- 데이터베이스에 얼굴 데이터 저장 없음
- 메모리 처리 후 즉시 삭제

### 접근 제어
- API 인증 (선택적)
- Rate Limiting
- 비정상 접근 차단

---

## 연예인 데이터

### 사용하는 데이터
- 공개된 연예인 프로필 사진
- 공개된 연예인 이름 및 정보

### 데이터 출처
- 공식 프로필
- 공개 라이선스 이미지

### 권리 보호
- 초상권 준수
- 상업적 이용 제한
- 요청 시 데이터 삭제

---

## 사용자 권리

### 데이터 삭제
- 분석 완료 후 자동 삭제됨
- 별도 삭제 요청 불필요

### 거부권
- 서비스 이용 거부 가능
- 웹캠 접근 거부 가능

### 문의
- 개인정보 관련 문의 지원
- 데이터 처리 내용 설명

---

## 쿠키 정책

### 필수 쿠키
| 쿠키 | 용도 | 만료 |
|-----|------|------|
| session_id | 세션 유지 | 브라우저 종료 시 |

### 선택 쿠키
- 분석 목적 쿠키 없음
- 광고 목적 쿠키 없음
- 추적 목적 쿠키 없음

---

## 제3자 제공

### 제공하지 않는 정보
- 사용자 얼굴 이미지
- 개인 식별 정보
- 분석 결과 (개인 식별 가능한 형태)

### 예외
- 법적 요구가 있는 경우
- 서비스 운영에 필수적인 경우 (익명화 후)

---

## 어린이 보호
- 만 14세 미만 이용 제한
- 법정 대리인 동의 필요

---

## 정책 변경
- 변경 시 사전 공지
- 주요 변경은 명시적 동의 요청

---

## 연락처
개인정보 관련 문의는 서비스 내 문의 기능을 이용해 주세요.
